
# Applied AI & ML Essentials – Capstone Project

## Overview

This repository contains my **Applied AI & ML Essentials Capstone Project**, completed in four parts. The project uses the **Ames Housing Dataset** to demonstrate the complete machine learning workflow, from data preparation to an LLM-powered feature.

## Project Structure

- **Part 1 – Data Preparation & Exploratory Data Analysis (EDA)**
  - Data loading and inspection
  - Data cleaning and preprocessing
  - Missing value handling
  - Outlier analysis
  - Exploratory visualizations

- **Part 2 – Supervised Machine Learning**
  - Regression model
  - Classification model
  - Feature encoding
  - Model evaluation
  - Performance comparison

- **Part 3 – Advanced Machine Learning**
  - Decision Tree
  - Random Forest
  - Gradient Boosting
  - Hyperparameter tuning
  - Feature importance analysis
  - Best model saved for deployment

- **Part 4 – LLM-Powered Feature**
  - Model prediction explanation
  - Structured JSON output
  - Prompt engineering
  - PII guardrails
  - End-to-end prediction explanation pipeline

## Dataset

**Ames Housing Dataset**

The dataset contains residential housing information and is used to predict housing prices using machine learning techniques.
